Google Sheets sync

Sync to a Google Sheet via a free Apps Script web app. One-time setup, ~3 minutes.

  1. Open Google Sheets → create a new spreadsheet
  2. Go to Extensions → Apps Script
  3. Delete placeholder code and paste the script below
  4. Click Deploy → New deployment → Web app
  5. Set Execute as: Me and Who has access: Anyone
  6. Authorize, copy the Web App URL, paste it below
